<h1>Quebec Mocktails Website</h1>

<p>
A responsive brand website developed for Quebec Mocktails Syrups to showcase products through an engaging and user-friendly interface.
</p>

<h2>Features</h2>

<ul>
  <li>Responsive design for desktop and mobile devices</li>
  <li>Interactive user interface using JavaScript</li>
  <li>Product showcase pages for different syrup flavours</li>
  <li>Clean and visually appealing layout</li>
  <li>Easy navigation and accessibility-focused design</li>
</ul>

<h2>Technologies Used</h2>

<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
  <li>Git</li>
  <li>GitHub</li>
</ul>
